Прогнозирование урожайности с использованием методов машинного обучения.

Описание задачи :

Найти зависимость между климатическими признаками и урожайностью разных культур. (Задача регрессии)

Признаки:
  • Средняя, максимальная и минимальная температура за день
  • Точка росы
  • Уровень моря и давление
  • Средняя и максимальая скорость ветра
  • Осадки за день
  • Наличие снега, града и дождя
    Для каждого месяца года взято среднее среди всех дней месяца и десяти точек измерения показание.
    Источник: Climate Data Online - Select Area
Метод решения:
  1. Нормализация данных (приведение значений к интервалу от -1 до 1) для лучшей работы математических моделей
  2. Обучение модели на части данных для каждой культуры
  3. Применение модели на оставшейся части данных
  4. Анализ применимости модели для данной культуры
  5. Выбор лучшей модели
  6. Прогноз с использованием модели
Модели машинного обучения:
  1. Эластичная сеть (ElasticNet)
  2. Эластичная сеть + PCA (Метод поиска главных признаков) (ElasticNet + PCA)
  3. “Kernel ridge” регрессия (а - alpha параметр регрессии) (Kernel Ridge Regression)
  4. “Kernel ridge” регрессия + PCA (Метод поиска главных признаков) (Kernel Ridge Regression + PCA)
  5. “Kernel ridge” регрессия + RBF ядро (Нелинейное преобразовние признаков для регрессии) (Kernel Ridge Regression + RBF Kernel)
  6. Регрессия деревьев решений (Decision Tree Regression)
  7. Композиция случайных деревьев решений (Random Forest Regression)
  8. Регрессия дополнительных деревьев решений (Extra Trees Regression)

Вывод:

Для решения этой задачи больше всего подходят: Kernel ridge (с rbf и линейныйм ядром) и ElasticNet
Методы основанные на деревьях решений не подходят в силу маленьких объемов данных
Но в некоторых случаях их предсказаия коррелируют с реальными данными.
Оценка точности: средняя
Так как набор данных небольшой, более продуктивные и точные методы машинного обучения применить сложно, так что возможно получить лишь примерную оценку.
Модели регрессии на данном датасете склонны занижать прогноз в половине случаев. Это может быть связанно так же с малым объемом данных и с зависимостью от предыдущего урожая, а не только от климатических признаков.
